# FastAPI-E-Commarce

This folder contains the full-stack e-commerce tutorial project split into a FastAPI backend and a React + Vite frontend.

## Project Layout

- [Backend](Backend/README.md) - FastAPI API, database models, migrations, and background task processing.
- [Frontend](Frontend/README.md) - React UI for customers and admins.

## What This Project Demonstrates

- User authentication and role-based access.
- Category, product, and review management.
- Background order status updates.
- FastAPI validation, dependency injection, and CORS integration.
- A Vite-powered frontend that talks to the API.

## Run Order

Start the backend first, then the frontend.

1. Install backend dependencies and configure the database and worker settings.
2. Run migrations and start the FastAPI server.
3. Install frontend dependencies and start the Vite dev server.

For detailed setup instructions, open the backend and frontend README files linked above.
