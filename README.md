# FastAPI Tutorial Workspace

This workspace collects several FastAPI learning projects, from small validation examples to a full e-commerce application with a React frontend.

## What's Here

- [FastAPI-E-Commarce](FastAPI-E-Commarce/README.md) - the full-stack e-commerce tutorial project.
- [fastapi_learning_project](fastapi_learning_project/README.md) - a compact project focused on path parameters, query parameters, request bodies, and dependencies.
- [deep_dive](deep_dive/README.md) - advanced examples that build dynamic Pydantic models at runtime.
- [SQL_MODEL](SQL_MODEL/README.md) - a larger SQLModel-based backend with auth, CRUD, migrations, and background task support.
- `main.py` and `Data_Annotation.py` - standalone learning scripts for FastAPI and Pydantic concepts.

## Suggested Starting Points

If you are exploring the workspace for the first time, start with the smaller examples first:

1. `main.py` for a minimal FastAPI introduction.
2. `fastapi_learning_project/README.md` for request handling and dependency injection.
3. `deep_dive/README.md` for dynamic model generation.
4. `FastAPI-E-Commarce/README.md` for the full-stack app.
5. `SQL_MODEL/README.md` for the SQLModel architecture and background jobs.

## Notes

- Each subproject is self-contained and may have its own dependencies and environment variables.
- Several folders intentionally mirror tutorial-style learning paths, so code may include commented explanations and experimental examples.
